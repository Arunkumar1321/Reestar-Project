# Reestar - Restaurant Website

Reestar is a simple restaurant website created using HTML and basic CSS.  
This project showcases a food menu with popular dishes and includes a contact form for users.

## Features
- Simple and clean restaurant layout
- Food menu with images and descriptions
- "Order Now" buttons for dishes
- Contact form with Name, Age, and Email fields
- Styled using basic CSS

## Technologies Used
- HTML
- CSS

## Purpose
This project was built as a beginner practice project to understand webpage structure, layout design, and basic styling in HTML and CSS.# Reestar-Project
